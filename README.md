# BSA_Requirements

Ознакомилась с требованиям к ПО, их видами, уровнями, связями и зависимостями. Составила таблицы по моделям as is и to be, построила контекстную диаграмму и таблицу входных/выходных потоков данных, выделила роли стейкхолдеров, их проблемы и потребности, выявила бизнес-требования продукта.
